# Tarea: Integración con JSON Server

# Descripción de la tarea

Esta tarea consistió en mejorar el formulario de creación de posts con las siguientes funcionalidades:

1. **Enlace para regresar**: Se agregó un botón para volver al listado de posts
2. **Validación de campos**: Se implementó validación para evitar envíos con campos vacíos
3. **Limpieza automática**: Los campos se limpian después de un envío exitoso
4. **Mensaje de confirmación**: Se muestra un mensaje de éxito después de crear un post
5. **Paginación**: Se implementó paginación en el listado de posts

# Explicación sobre CORS

# ¿Qué es CORS?
CORS (Cross-Origin Resource Sharing) es un mecanismo de seguridad que permite o restringe recursos solicitados desde un dominio diferente al que sirvió el primer recurso.
